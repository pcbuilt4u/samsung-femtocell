> OMAP171x - 220 MHz ARM926EJ-S + C55x DSP, Low-voltage 90 nm technology

# Web Login #
The device requests an IP via DHCP.  Once you find it's address, you can access it on port 80, with:
```
username: admin
password: ubicell
```
